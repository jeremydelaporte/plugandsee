Purpose:
A program to implement Analog to  Digital Converter (ADC) example on LPC13Uxx

Note:
Tested after building using LPCXpresso on LPC1347 

Running mode:
* Build, Debug
* Setup UART for 11500, 8 data bits, no parity, 1 stop bit
* Vary the Trim Pot on board  
  ** AD6 (Channel 6) on IAR LPC-1347-SK
